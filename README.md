# homebrew-tap

사내 CLI 도구 Homebrew tap.

## 설치

```bash
brew tap michael-lim-watcha/tap
```

## 도구

| 이름 | 설명 | 설치 |
|------|------|------|
| watcha-pedia | 왓챠 피디아 CLI (guinness API 래핑) | `brew install michael-lim-watcha/tap/watcha-pedia` |
| khao | 사내 IDP CLI | `brew install michael-lim-watcha/tap/khao` |
